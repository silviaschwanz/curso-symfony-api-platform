Instruções para configurar xdebug no windows:

No terminal linux wsl executar o seguinte comando --> echo $(grep nameserver /etc/resolv.conf | cut -d ' ' -f2)

o número IP mostrado deverá ser colocado nas configurações de host do xdebug, mas ele pode mudar cada vez que a máquina for ligada
